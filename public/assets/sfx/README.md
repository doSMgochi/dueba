# Yacht SFX assets

Place the selected sound-effect files in this directory with these exact names:

- `yacht-dice-shake-loop.mp3`
- `yacht-dice-roll.mp3`
- `yacht-cheer.mp3`

Suggested free sources:

- Dice roll/shake: Pixabay, `dice` by JayeWilde (Freesound), https://pixabay.com/sound-effects/film-special-effects-dice-65736/
- Additional dice search: https://pixabay.com/sound-effects/search/roll%20dice/
- Cheer: Mixkit crowd sound effects, e.g. `Cheering crowd loud whistle`, `Huge crowd cheering victory`, or `Male crowd cheering short`, https://mixkit.co/free-sound-effects/crowd/

Check each downloaded file's license page before shipping. The game code fails silently if a file is missing, so development can continue before final audio files are added.
